Suppression de la base $database

Vous allez supprimer complètement la base $database.

Le répertoire contenant la base de données va être supprimé. Tous les fichiers et tous les dossiers présents dans ce répertoire vont être supprimés, même s'il s'agit de fichiers étrangers à la base de données.

La suppression est irréversible : vous ne pourrez recréer la base de données que si vous disposez d'un dump à jour de la base $database.
