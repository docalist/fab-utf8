Modification du schéma de la base $database

La modification de la structure d'une base de données se fait en appliquant un nouveau schéma à une base de données (en général, il s'agit d'une nouvelle version du schéma utilisé pour créer la base, mais ce n'est pas obligatoire).

Une confirmation vous sera demandée avant d'appliquer le nouveau schéma. Cette page de confirmation liste toutes les modifications apportées entre le schéma actuel de la base de données et le schéma choisi, ce qui vous permet de vérifier que c'est bien ce que vous voulez faire.
