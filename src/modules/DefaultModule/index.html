Bienvenue !

Votre nouvelle application vient d'être créée.

Pour le moment, elle utilise la configuration par défaut.

Configuration

Le fichier de configuration principal s'appelle general.config et il a été créé pour vous dans le répertoire /config de votre application. Par la suite, vous allez créer de nouveaux fichiers de configuration pour chacun des modules de votre application.

Consultez la documentation pour obtenir des informations supplémentaires sur les fichiers de configuration.

Look de votre application

Un thème a été créé pour vous. Il s'appelle "minimalistic" et il se trouve dans le répertoire /themes/ de votre application. Deux dispositions différentes sont disponibles : default.html qui a une barre latérale à gauche, et nomenu.html qui n'en a pas (exemple).

Création des bases de données

Vous souhaitez probablement créer des bases de données pour votre application : pour chaque base de données, commencez par créer son schéma (c'est-à-dire sa structure) en utilisant l'éditeur intégré puis créez la base de données à partir de ce schéma.

Création de nouveaux modules

Fab dispose de quelques modules prêts à l'emploi que vous n'avez plus qu'à configurer et à personnaliser. Vous pouvez également fabriquer vos propres modules en créant un nouveau répertoire dans le répertoire modules de votre application.

Consultez la documentation pour plus d'informations.

Templates

Fab dispose d'un système de templates qui permet d'avoir une séparation nette entre la présentation du site et le code PHP sous-jacent. Les templates peuvent être utilisés pour générer n'importe quelle sortie au format texte (texte brut, xml, html, rtf, etc.) Ils constituent une solution simples, rapide, évolutive et modulaire pour toutes les pages de votre application.

Consultez la documentation pour plus d'informations.
